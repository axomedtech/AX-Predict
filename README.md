## 📌 About

AX-Predict is an AI-based healthcare prediction system developed under the AXOMED project. The system helps analyze patient health data and predict possible medical risks using Machine Learning and real-time monitoring concepts.

---

## ✨ Features

- ✅ Patient Data Management
- ✅ AI Health Risk Prediction
- ✅ Flask Web Dashboard
- ✅ Database Integration
- ✅ Real-Time Monitoring Ready
- ✅ Clean Project Structure
