# 📌 About

AX-Predict is an AI-powered healthcare prediction system developed under the AXOMED project.  
The platform analyzes patient health data and predicts possible medical risks using Machine Learning and smart monitoring technologies.

---

# ✨ Features

<div align="center">

| Feature | Description |
|---|---|
| 🤖 AI Prediction | Predicts patient health risks |
| 📊 Dashboard | Flask-based web dashboard |
| 🗂 Patient Manager | Stores and manages patient data |
| 💾 Database | Database integration support |
| ⚡ Real-Time Ready | Supports live monitoring systems |
| 🔒 Clean Architecture | Organized and scalable structure |

# ⚙️ Installation

```bash
git clone https://github.com/axomedtech/AX-Predict.git

cd AX-Predict

pip install -r requirements.txt

python app.py
```
